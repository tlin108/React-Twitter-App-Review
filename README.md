### How to run app
```sh
1. git clone this repo
2. cd into repo directory
3. npm install or yarn install
4. Please either supply an API key directly by swapping out the API_KEY constant in getAPIroute method
   or create a config folder and in the folder, create a config file with module exporting the API key
5. npm start or yarn start
6. open [http://localhost:3000/](http://localhost:3000/) to see app
```
